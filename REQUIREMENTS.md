This project seeks to create a useable, user friendly, terminal centric Linux experience, based on NixOS.

Born from the vision of combining the power of NixOS with the elegance of terminal-based computing, nixmywindows represents a return to the essence of what makes Linux powerful - direct, efficient, and completely under your control.

## Features

1. Based on NixOS and Nix Flakes
2. ZFS with encryption as the underling file system
3. Terminal only - no Wayland or X11
