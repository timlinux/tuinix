# NIX MY WINDOWS
## A Pure Terminal Based Linux Experience

This project seeks to create a useable, user friendly, terminal centric Linux experience, based on NixOS.

## Features

1. Based on NixOS and Nix Flakes
2. ZFS with encryption as the underling file system
3. Terminal only - no Wayland or X11
